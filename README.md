List keys
---

* `list-keys -t vi-copy`

Sessions
---

* `tmux list-sessions`
* `tmux attach [session_name]`

Panes
---

* `:setw synchronize-panes on|off` - send commands to all panes
* `C-b + {` 
* `C-b + }` - Shift panes left or right
* `C-b + z` - Zoom/ unzoom pane

Windows
---

* `C-b + (`
* `C-b + )` - Cycle through windows
